Recent Administration Changes

9 September 2023

Team names, student names, school names, mentor names and some other fields are now restricted to common ASCII characters.

Platform Category has been added and can be used to group different Hardware Platforms and Software Platforms. New option to filter by Hardware and Software Platform on teams list page.

Max number of teams per division now checked when a mentor attempts to move a team between divisions. Coordinators can still override this in the admin portal.

Upload file type extension now case insensitive.

Upload submission date must now be on or after registration close date and on or before competition start date.

Mentors can now view published events before registration opens, such as if they click on a link from the website. These events will not be listed on the dashboard until registration opens.

6 September 2023
