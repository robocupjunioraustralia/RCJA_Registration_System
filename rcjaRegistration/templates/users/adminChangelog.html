Recent Administration Changes

1 February 2025

Removed the school abbreviation field as it was not being used.

Added ability to set max number of registrations for workshops. The same options for teams are available.

Updates to the Association membership portal.

22 January 2025

Can now set event dates and competition payment amount to blank values, which shows as TBC to mentors. Registration doesn't open until all fields are filled out.

Can create events in the CMS and link to the registration system from within the registration system. See CMS documentation for more information.

There is a new interface for mentors to copy teams from one event to another, which allows copying events from previous years as well.

New ability to override which invoice a team or workshop attendee is linked to, to enable registration affiliation in the CMS to be different to the billing contact.

11 September 2023

Can now search by invoice number, PO number, invoice amount, amount due, and amount paid on invoice page.

Added links to teams and invoices to event details page.

10 September 2023 - Invoice Improvements

Invoice page is much, much faster to load.

Can now filter invoices based on amount due and invoice amount - $0 invoices are now hidden by default. Can also sort by invoice amount, amount paid and amount due - click the column header.

Invoice date now defaults to payment due date. If the mentor views the invoice before the payment due date the date updates to the current date the first time the mentor views the invoice. Invoice date can still be changed manually.

Negative amounts due now display as $0.

9 September 2023

Team names, student names, school names, mentor names and some other fields are now restricted to common ASCII characters.

Platform Category has been added and can be used to group different Hardware Platforms and Software Platforms. New option to filter by Hardware and Software Platform on teams list page.

Max number of teams per division now checked when a mentor attempts to move a team between divisions. Coordinators can still override this in the admin portal.

Upload file type extension now case insensitive.

Upload submission date must now be on or after registration close date and on or before competition start date.

Mentors can now view published events before registration opens, such as if they click on a link from the website. These events will not be listed on the dashboard until registration opens.
